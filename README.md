# My Portfolio

Personal portfolio website. Built with Nuxt 4, Tailwind CSS, and GSAP.

## Stack

- Nuxt 4 + Vue 3
- Tailwind CSS v4
- GSAP animations
- i18n — English, Russian, Uzbek
- Dark / Light mode
- Contact form → Telegram notification

## Run locally

```bash
npm install
npm run dev
```

## Environment

Create `.env` from `.env.example` and fill in:

```
NUXT_PUBLIC_SITE_URL=https://nizomovnurmuhammad.uz
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```
